Custom SQL Reformatter

This project is a custom SQL formatting tool built using Python and Streamlit. It allows users to paste raw SQL and apply formatting rules based on team-specific standards, such as:

  • Lowercase SQL keywords
  • Leading commas in select clauses
  • PascalCase for field and CTE names
  • CASE/WHEN/THEN alignment
  • Strict no-renaming or reflowing logic

It’s useful for teams with strict SQL style guides or analysts who want consistent formatting across queries and PRs.
